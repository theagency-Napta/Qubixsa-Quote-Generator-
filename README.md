# نظام عروض الأسعار - Qubixsa Quote System

نظام متكامل لإدارة وتسعير وعرض عروض الأسعار التفاعلية لعملاء **كيوبيكس السعودية (QUBIXSA)** مع لوحة تحكم مبيعات إدارية مدمجة.

---

## 📁 هيكلية المشروع (Project Architecture)

يحتوي المشروع على الملفات الرئيسية التالية:

1. **`generator.html` (منشئ عروض الأسعار)**:
   * المنصة الرئيسية لمندوبي المبيعات لإنشاء وتعديل وتصميم عروض الأسعار.
   * يدعم جلب العملاء والمنتجات مباشرة من قاعدة بيانات Supabase.
   * يحتوي على **نظام تتبع التكاليف والأرباح (COGS)** المحمي برمز مرور (`1055`) لمراقبة هوامش الأرباح وتقديم تحذيرات مرئية عند انخفاض الهامش عن 25%.
   * يتيح نشر ومزامنة العرض مباشرة إلى السحاب بضغطة زر.

2. **`portal.html` (بوابة العميل التفاعلية)**:
   * الرابط المباشر الذي يتم مشاركته مع العميل لمشاهدة عرض السعر.
   * يتيح للعميل استعراض الخيارات (Option A / Option B) وتفعيل/تعطيل البنود الاختيارية وتحديث الإجمالي والضريبة تلقائياً.
   * يدعم التوقيع الرقمي الإلكتروني (E-Signature) للاعتماد المباشر وتحديث حالة العرض في قاعدة البيانات مع إرسال إشعارات فورية عبر الويب هوك.

3. **`admin.html` (لوحة تحكم الإدارة وتحليل المبيعات)**:
   * لوحة تحكم إدارية محمية برمز المرور (`1055`) تتيح للمدراء مراقبة:
     * مؤشرات الأداء الرئيسية (KPIs): إجمالي الأنبوب المبيعاتي، الصفقات المغلقة، صافي الأرباح ومعدل التحويل.
     * رسوم بيانية تفاعلية (Chart.js) لتوزيع الحالات وحجم المبيعات الشهري.
     * جدول متابعة وسجل لجميع عروض الأسعار مع إمكانية فتح البوابة، نسخ الرابط أو الحذف.

## 🛠 إعداد قاعدة البيانات (Database Setup - Supabase)

لتفعيل كامل ميزات تتبع هوامش الأرباح والتحليلات الإدارية، يرجى تشغيل النص البرمجي (SQL) التالي في **محرر الاستعلامات (SQL Editor)** الخاص بـ Supabase لإضافة الأعمدة الإحصائية المطلوبة:

```sql
-- 1. إضافة أعمدة التحليل الإحصائي لجدول عروض الأسعار
alter table public.quotes 
  add column if not exists total_price numeric default 0,
  add column if not exists total_cost numeric default 0,
  add column if not exists margin_percent numeric default 0;

-- 2. تحديث السجلات القديمة تلقائياً (تجنباً لقيم Null)
update public.quotes 
set 
  total_price = coalesce((payload->'grand')::text::numeric, 0),
  total_cost = coalesce((payload->'subtotal')::text::numeric, 0) * 0.7, -- تقدير تكلفة افتراضي بنسبة 70%
  margin_percent = 30
where total_price = 0;
```

## 🌐 الاستضافة والتفعيل (Deployment via GitHub Pages)

بما أن النظام يعتمد بالكامل على تقنيات الويب الثابتة (HTML / CSS / JS) للتواصل المباشر مع Supabase، يمكن استضافته مجاناً وبسرعة عبر **GitHub Pages**:

1. اذهب إلى إعدادات المستودع الخاص بك على GitHub (**Settings**).
2. اختر قسم **Pages** من القائمة الجانبية.
3. تحت خيار **Build and deployment**، اختر الفرع الرئيسي `main` كمصدر للرفع (Source) ثم اضغط **Save**.
4. خلال دقائق، سيقوم GitHub بتوفير روابط مباشرة للملفات الثلاثة، على سبيل المثال:
   * بوابة الإدارة: `https://[username].github.io/[repo-name]/admin.html`
   * منشئ عروض الأسعار: `https://[username].github.io/[repo-name]/generator.html`

## 🔒 حماية وخصوصية البيانات

* **الخصوصية**: يوصى بشدة بالإبقاء على هذا المستودع **خاصاً (Private)** لحماية رمز مرور الإدارة ومفاتيح الربط البرمجية المدمجة.
* **سياسات الأمان (RLS)**: تم تكوين سياسات الأمان في Supabase لضمان قدرة البوابة والمنشئ على إدراج وتعديل البيانات بأمان دون ثغرات أمنية.
