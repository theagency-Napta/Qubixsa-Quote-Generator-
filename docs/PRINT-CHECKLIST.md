# Qubixsa — Print / PDF checklist

Use **Chrome** (recommended) when saving quotes from `qubixsa-quote-generator.html`.

## Before printing

1. Complete the **Ready to send** checklist (all green ✓).
2. Review the live preview (full width, Arabic readable).
3. Confirm quote number appears in the bank note.

## Print dialog settings

| Setting | Value |
|---------|--------|
| Destination | Save as PDF |
| Paper | A4 |
| Margins | Default |
| Scale | **100%** (not “Fit to page”) |
| Background graphics | **ON** |

## After export

- Open the PDF and confirm orange headers and table stripes are visible.
- Confirm content uses the **full page width** (no large empty strip on one side).
- Confirm **البنود والشروط التفصيلية** appears if enabled.

## Export HTML (alternative)

Use **Export HTML** in the generator for a standalone file (no form panel) — open in browser and print with the same settings.
